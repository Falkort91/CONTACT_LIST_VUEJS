# Démo: [Contact_list_vueJs](https://falkort91.github.io/CONTACT_LIST_VUEJS/)

Descirption : Un simple contact list en vuejs ou il est possible de : 
- d'ajouter un contact
- modifier un contact
- supprimer un contact
- rechercher un contact
- filtrer les contacts par ordre alphabétique
- (avec un compteur dynamique du nombre de contact)
